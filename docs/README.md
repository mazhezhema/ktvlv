## 📚 文档分类说明

### 🔧 fixes/ - 问题修复与诊断
记录项目开发过程中遇到的各种问题及其修复方案，包括：
- 黑屏问题诊断与修复
- 异常处理修复
- 初始化顺序问题修复

### 👀 reviews/ - 代码审查报告
代码审查和评估报告，包括架构评估、代码质量分析等。

### 🏗️ build/ - 构建相关
构建系统相关文档，包括：
- 构建流程说明
- 构建最佳实践
- 快速构建指南

### 🏛️ architecture/ - 架构设计
系统架构设计文档，包括：
- 整体架构设计
- 模块架构设计
- UI 架构设计

### 🎨 design/ - 系统设计
各功能模块的详细设计文档，包括：
- 许可证管理系统
- HTTP 客户端设计
- 下载与存储设计
- 事件系统设计

### 📖 guides/ - 开发指南
开发指南和最佳实践文档，包括：
- 编码规范
- 开发模板
- 移植指南
- UI 最佳实践

### 🔌 api/ - API 文档
API 接口文档，包括服务器 API 列表等。
