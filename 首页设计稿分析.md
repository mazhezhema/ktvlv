# 首页设计稿详细分析

> **分析日期**：2025-12-26  
> **设计稿来源**：用户提供的真实首页设计图

## 一、设计稿布局结构

### 1.1 顶部栏（Top Bar）

```
┌─────────────────────────────────────────────────────────────────┐
│ [Logo: 雷石官方正版] [首页] [猜你喜欢] [历史记录] [我的收藏] [VIP会员中心] │
└─────────────────────────────────────────────────────────────────┘
```

**元素说明：**
- **左侧Logo**：眼睛图标 + "雷石官方正版" 文字
- **导航标签**（从左到右）：
  - "首页"（蓝色高亮，当前选中）
  - "猜你喜欢"（未选中）
  - "历史记录"（未选中）
  - "我的收藏"（未选中）
- **右侧按钮**："VIP 会员中心"（渐变按钮）

**注意**：根据用户要求，**不实现"猜你喜欢"和"我的收藏"**

**布局说明：**
- **左侧视频播放器**：大矩形，紫色背景，显示歌手图片
- **中间视频播放器**：大矩形，深色背景，显示MV画面，右上角有微信扫码点歌二维码
- **右侧排行榜区域**：两个垂直排列的排行榜面板
  - **上方**："情歌对唱榜"（浅蓝色背景，3首歌曲）
  - **下方**："抖音热唱榜"（粉色背景，3首歌曲）

### 1.3 主内容区（下半部分）

```
┌─────────────────────────────────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ │
```

**5个大按钮/区域**（水平排列）：
1. **"歌手"**：粉色到紫色渐变，右侧有3个圆形头像
2. **"排行榜"**：粉色到红色渐变，带奖杯图标
3. **"分类"**：蓝色到浅蓝渐变，带文件夹图标
4. **"热歌榜"**：粉色到红色渐变，带红色星星图标
5. **"免费专区 即刻开唱"**：蓝色到紫色渐变，带星爆图标，两行文字

### 1.4 底部控制栏

## 四、布局尺寸估算

假设屏幕分辨率：1280x720

- **顶部栏**：高度 50px
- **主内容区上半部分**：高度约 400px
  - 左侧视频播放器：宽度约 400px
  - 中间视频播放器：宽度约 400px
  - 右侧排行榜：宽度约 480px，每个排行榜高度约 200px
- **主内容区下半部分**：高度约 150px
  - 5个大按钮：每个宽度约 256px
- **底部控制栏**：高度 80px

## 五、颜色方案（从设计稿提取）

- **紫色系**：`#5F4B9A`（深紫）、`#7C6BCB`（浅紫）

- **蓝色系**：`#4F7BFF`（蓝色）、`#6E8FFF`（浅蓝）
- **排行榜背景**：
  - 情歌对唱榜：浅蓝色 `#E6F3FF`
  - 抖音热唱榜：粉色 `#FFE6F0`
